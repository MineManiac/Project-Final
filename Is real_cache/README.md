The objective of the game is to evade all Coronavirus, getting as many masks as possible n\

## Controls
Key_UP -> Goes Up n\
Key_DOWN -> Goes Down n\
Key_LEFT -> Goes Left n\
Key_RIGHT -> Goes Right n\
